# Generation-of-hypotheses
Исследовательский проект по дисциплине "проектный практикум"

Тема: Генерация гипотез на основе массива научных статей с помощью искусственного интеллекта

Цель: Провести исследования на то, чтобы было возможно из отфильтрованных статьей сделать генерацию гипотез про "металлургию", однако мы же будет делать не только исследования а полноценный веб-сайт.
