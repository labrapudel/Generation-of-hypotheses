# Generation-of-Hypotheses
Исследовательский проект по дисциплине "Проектный практикум"

## Тема
Генерация гипотез на основе массива научных статей с помощью искусственного интеллекта

## Цель
Разработать веб-приложение, которое позволяет:
- Анализировать научные статьи из различных источников (PDF, URL);
- Генерировать оригинальные, проверяемые и аналитически обоснованные гипотезы;
- Работать не только с тематикой "металлургия", но и с другими научными направлениями;
- Обеспечивать удобный интерфейс для исследователей, сокращая время на обработку литературы и подготовку идей для экспериментов.

## Основные функции
- Извлечение текста из PDF и веб-страниц;
- Генерация гипотез с использованием ИИ (Groq API);
- Хранение и управление результатами через базу данных (SQLAlchemy);
- Интерактивный веб-интерфейс для удобного использования.
